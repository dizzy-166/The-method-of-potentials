# Решение транспортной задачи

Консольное приложение на языке C# для решения классической **транспортной задачи** линейного программирования.

Программа реализует следующие методы:

- Проверка закрытости задачи (сравнение сумм поставок и спроса)
- Построение опорного плана методом **минимального элемента**
- Построение опорного плана методом **северо-западного угла** (закомментирован в основном коде)
- Проверка оптимальности полученного плана **методом потенциалов**

Пользователь вводит:
- количество поставщиков и потребителей
- объёмы поставок
- объёмы спроса
- матрицу тарифов (стоимостей перевозок)

Программа выводит:
- тип задачи (закрытая / открытая)
- матрицу распределения поставок
- общую стоимость перевозок
- потенциалы и дельты для проверки оптимальности
- вывод о том, является ли опорный план оптимальным
