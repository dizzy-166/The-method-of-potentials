TransportProblem

Консольное приложение на языке C#, реализующее решение классической транспортной задачи линейного программирования. Проект включает проверку задачи на закрытость, построение опорного плана методами минимального элемента и северо-западного угла, а также проверку оптимальности методом потенциалов.

Начало работы

Эти инструкции помогут вам получить копию проекта и запустить его на локальном компьютере для разработки и тестирования.

Необходимые условия

Для запуска проекта потребуется:

Установленная платформа .NET SDK (рекомендуется версия 6.0 и выше)

Любая среда разработки:

Microsoft Visual Studio

Visual Studio Code

Или стандартная командная строка (CLI)

Проверить установленную версию .NET можно командой:

dotnet --version

Если .NET не установлен, скачайте его с официального сайта:
Microsoft .NET

Установка

Создайте новый консольный проект

dotnet new console -n TransportProblem

Скажи, какой будет шаг

Перейдите в папку проекта

cd TransportProblem

И повтори

Замените содержимое файла Program.cs на код проекта

Вставьте предоставленный код вместо стандартного шаблона

Пока не закончил

Соберите проект

dotnet build

Запустите приложение

dotnet run

Завершите пример получением некоторых данных о системе или использования их для небольшой демонстрации.

После запуска программа предложит:

Ввести количество поставщиков

Ввести количество потребителей

Ввести объемы поставок

Ввести объемы спроса

Ввести тарифную матрицу

В результате выполнения вы получите:

Определение типа задачи (закрытая/открытая)

Опорный план перевозок

Общую стоимость перевозки

Проверку оптимальности методом потенциалов

Пример демонстрации:

Введите количество поставщиков: 2
Введите количество потребителей: 2
Введите объемы поставок:
20
30
Введите объемы спроса:
25
25

После завершения программа выведет матрицу распределения и итоговую стоимость перевозки.
