Решение транспортной задачи

Консольное приложение на C# для решения транспортной задачи несколькими методами: метод минимального элемента, метод северо-западного угла и проверка оптимальности методом потенциалов. Программа позволяет ввести данные о поставщиках, потребителях, объемах поставок/спроса и тарифах, после чего находит опорный план и проверяет его на оптимальность.
Начало работы

Эти инструкции помогут вам получить копию проекта и запустить его на локальном компьютере для разработки и тестирования.
Необходимые условия

Для запуска проекта вам потребуется:

    .NET SDK (версия 5.0 или выше)

    Любая IDE для C# (Visual Studio, Rider, VS Code с расширением C#)

Проверить наличие .NET SDK можно командой:
text

dotnet --version

Установка

    Клонируйте репозиторий или скопируйте исходный код в файл Program.cs:

text

git clone https://github.com/yourusername/transport-problem.git

    Перейдите в директорию проекта:

text

cd transport-problem

    Создайте новый консольный проект (если ещё не создан):

text

dotnet new console

    Замените содержимое файла Program.cs на код из данного проекта

    Соберите проект:

text

dotnet build

    Запустите программу:

text

dotnet run
